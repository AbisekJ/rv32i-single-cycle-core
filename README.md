# rv32i-single-cycle-core

## Overview
32-bit pipelined RISC-V processor designed in Verilog and implemented on Arty A7 FPGA.

## Features
- Single Cycle
- Base Integer Type
- All 37 Instructions

## Tools Used
- Verilog
- Vivado
- Arty A7

## Folder Structure
rtl/ : source files  
tb/ : testbench  

## Results
Validated at 50 MHz.
